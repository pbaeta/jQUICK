jQUICK

jQuery User Interface and Controls Kit

 

jQUICk (or QUIC for short) is a UI/UX suite of jQuery pugins used to create and style user interaction widgets and elements like: butons, form inputs and other controls.
This suite can be used to style and boost a complex web app or just to add a simple form control to a site. Likewise the level of knowlege needed to implement this suite is minimal (though at performance expense) but for the savy developer, more effecient and flexible methods are provided.

Usage

To use JQUIck first you'll need to include the JS and CSS files on the head section of your HTML and after that set the classes of elements to the fallowing examples
